# Geological-Intelligence

Repositório para organização e versionamento de projetos relacionados à inteligência geológica.

## Estrutura Inicial
- Dados e scripts de competições Kaggle
- Notebooks de experimentos
- Outros arquivos relevantes

## Como usar
1. Clone este repositório:
   ```bash
   git clone https://github.com/angualberto/Geological-Intelligence.git
   ```
2. Adicione seus arquivos e scripts.
3. Faça commits e push normalmente.
